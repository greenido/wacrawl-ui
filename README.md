# WaCrawl Dashboard

Local WhatsApp analytics dashboard for the SQLite archive generated by `wacrawl`.

## Requirements

- Node.js 20+
- A WaCrawl archive at `~/.wacrawl/wacrawl.db`

## Development

```bash
npm install
npm run dev
```

The API runs on `http://127.0.0.1:3001` and the Vite app runs on `http://localhost:5173`.

## Configuration

The API opens the database in readonly mode. Override the default database path with:

```bash
WACRAWL_DB=/path/to/wacrawl.db npm run dev -w @wacrawl/api
```
